# Mini Todo App

A simple todo application built with vanilla JavaScript, evolving from basic DOM manipulation to a state-driven architecture.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks with a dedicated button
- Automatically update task numbers
- State management using a JavaScript array
- UI rendering based on application state

## Project Structure

This project demonstrates the transition from direct DOM manipulation to a more structured approach using a central state (`todos` array) and a render function.

## Practice Focus

In this project, I practiced:

- dynamic DOM creation and updates
- handling user input and events
- managing application state with arrays and objects
- implementing a render function (`renderTodos`)
- separating data from UI logic
- updating UI based on state changes
