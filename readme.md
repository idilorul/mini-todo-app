# Mini Todo App

A simple todo application built with vanilla JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks with a dedicated button
- Automatically update task numbers after deletion

## Project Structure

This project focuses on core DOM interaction and dynamic task management in a minimal interface.

## Practice Focus

In this project, I practiced:

- creating and appending DOM elements dynamically
- handling user input
- working with event listeners
- separating text and button elements for cleaner updates
- updating task order after deletion

## Technologies

- HTML
- CSS
- JavaScript
